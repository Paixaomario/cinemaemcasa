🎯 3 PASSOS PARA COMEÇAR:

1️⃣  SETUP (5 minutos):
    ├─ npm install (se não tiver)
    ├─ Gere VAPID keys: https://vapidkeygen.appspot.com/
    ├─ Adicione em .env.local
    └─ Execute: npm run dev

2️⃣  DATABASE (5 minutos):
    ├─ Abra Supabase SQL Editor
    ├─ Cole: supabase/migrations/017_webos_push_notifications.sql
    ├─ Execute (deve criar 4 tabelas)
    └─ Verifique RLS ativado

3️⃣  BUILD & INSTALL (15 minutos):
    ├─ npm run build:webos (gera arquivo)
    ├─ scp para TV (via SSH)
    ├─ Descompacte em /media/developer/apps/
    └─ Reinicie TV
